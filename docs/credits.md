## Author

Resonix was designed, written, and is actively maintained by the VeDevelopment team. Every layer —
the two-stage acoustic propagation engine, the frequency-dependent material absorption model, the
sound buffer and relevance window, the occlusion cache, the binary serialization protocol,
the clock drift corrector, and the ResonixNet middleware — is original work.

## Dependencies

Resonix ships with one internal dependency:

**VeSignal** — a high-performance, type-safe signal implementation for Roblox Luau, written by
VeDevelopment. It uses connection pooling, scratch arrays, and thread recycling to minimise GC
pressure. Used throughout ResonixNet for server and client event routing.
